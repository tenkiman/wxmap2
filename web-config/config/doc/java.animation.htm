PCMDI WXMAP Java Animation --

Warnings and Suggestions

This Documentation is under active development, please be patient.....

First things first. Your browser must be JAVA aware and you'll need lots of memory before these movies will work... Further, your OS must be 32-bit, so you MS Windows 3.1 (16-bit) folks are out of luck

While I'm not I big fan of animation for weather foreasting and believe animation has been grossly oversold, movies can be useful for getting an overiew of the what's happening. I'll typically look at an animation to identify the "interesting" forecast time and then go to the static maps. I first considered animated gifs for my movies, but did not go this route for three reasons: 1) Netscape loops the gif images over the network whereas Internet Explorer 3.0 (IE3.0) brings the file over and then loops; 2) I would have to keep two copies of the images; and 3) I could not let the user stop/start the animation. Java seemed like the obvious thing and COLA got a slick animation going...

With the help of the COLA Weather Maps webmaster, Dr. Paul Dirmeyer, I got the java applet used in their 30-day movie of Northern Hemisphere 500 mb heights and vorticity running in my web. You can access the COLA movie be clicking here, but be careful as it will take a while to load...

I initially tested my animations in IE3.0 and under this browser, it was possible to resize the images in the animation. That is, I could animate at a different size than the original images. Unfortunately, Netscape 3.0 does not support this capability, so when I changed my movies to maintain resolution it started working under Netscape. However, the performance under IE3.0 is better...

I've tested the animations under the following systems.

Machine/OS Browser Comments
Inter Pentium/WinNT4.0 Internet Explorer 3.0 Works great, best performance
Mac PowerPC, etc. Netscape 3.0 DOES NOT WORK WITHOUT 16+ MB May load only a few of the images and stop
Inter Pentium/Win95 Internet Explorer 3.0 As good as WinNT
Inter Pentium/Win95 Netscape Gold 3.0 Work nearly as well as IE3.0, but speed is slower (better caching under IE)
Sun Sparc 10/Sunos 4.1.3 Netscape 3.0 Works, quality of image not as good as in IE3.0
Sun Sparc 10/Solaris 2 Netscape 3.0 As good as Sunos
SGI Indy /Irix 5 Netscape 3.0 About same as Sunos

Please report your experience so I can fill in the table, but for the future, I'm looking for alternatives to this rather old java applet. Your suggestions would be appreciated
