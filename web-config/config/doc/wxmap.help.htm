Hints! -- suggestions for solving common problems

When in doubt... hit your browser's Refresh or Reload button

The dynamic nature of the documents in this web, and the need to update graphics, may cause an image or document (you'll get a 404 error) to either partially load or not load at all (it might be generating). Using "reload" may help.

Using the "back" and "forward" browser buttons

The "back" button is very handy for recovering from erroneous clicks. I use it all the time.

I also find these browser buttons useful to "flicker" animate maps, e.g., compare the t=72 h forecast from the NCEP AVN versus the FNMOC NOGAPS model. However, there are model buttons to do the same thing, but sometimes this can more convenient.

Shutdown your Browser...

Java animations and loading lots of maps may fill up the cache in your browser causing slow performance. The only way I know to free the memory is to shutdown the browser.

The following table gives an example of how you might use the PCMDI WXMAP Web.

Typical Session using the PCMDI WXMAP Web
Action What Happens
Go to the Home Page The time when the web was created, the base DTG of the maps, and the current DTG are displayed. You're asked to select an area.
Click on the CONUS Area Icon You'll go to the home page for that area. Again the DTG of the maps is displayed, but now you'll find a table with navigation buttons to go to other available model home pages. The number of available models depends on the time of day.
Click on the AVN Home button The model, base DTG of the maps and the active area (the one you selected at the step above) is displayed in title section, but now you'll see two table to work with. The first is where you select the maps or movies you want to see for that model. The second contains navigation buttons to go to other available model home pages. The number of available models depends on the time of day.
Click on the green 24-h 500 mb forecast If the button is NOT green, then it means the map was not produced for some reason.
The map will appear with two rows of navigation buttons to go to other available model home pages. Again, depending on model and forecast availability, a variety of actions are possible.
Click on the NGP button This allows you to intercompare the AVN forecast with that from the Navy model (NOGAPS or NGP). The only difference. Even the data grids are the same (1 deg).
Click on +12 button This advances you 12 hours into the future. You can move backwards and forewards in time.
Click on All Maps button You see something interesting and now want to look at all the maps for the forecast time. All the maps are laid out in a single document, you scroll down and see a big rain storm forming over the midwest....
Click on Precip map This takes you to the html document with the graphic AND the navigation buttons. You are now curious if the big storm is only seen in this run of the model, i.e., has the model been consistently forecasting this storm.
Click on Prev Run -12 h button The the forecast made 12 h earlier (e.g., the 36-h forecast made by 96102700 run), but verifying at the time of the previous map (e.g., the 24-h forecast made by the 96102712 run) is displayed. You can keep going backwards (or forward) until there are no verifying forecasts available.

Here's a list of the less obvious terms and acroynms you'll run across my web.

Term Definition
AVN The "Aviation" or short integration (0-72 h) of the NCEP MRF model made every 00, 06, 12 and 18 UTC.
DTG The"Date Time Group" or how I represent time. The format is (yymmddhh or 19yymmddhh) where yy is the year, mm the month number, dd the day number and hh the UTC hour.
FNMOC The U.S. Navy Fleet Numerical Meteorology and Oceanography Center, Monterey, CA
MRF The NCEP Medium Range Forecast gloabl model developed by the NCEP Environmental Modeling Center. Also refers to the long integration of the MRF made every 00 UTC.
NCEP The National Centers for Environmental Prediction
NGP
NOGAPS 		The Navy Operational Global Atmospheric Prediction System developed by Naval Research Lab, Monterey
NWP Numerical Weather Prediction, i.e, the computer models of the atmosphere
